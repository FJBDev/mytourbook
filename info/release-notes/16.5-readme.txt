Version 16.5                                               ?. May 2016

  

New
===	

* Table/Tree Header Context Menu

  - Hide hovered column
  - Set value format for the hovered column, e.g. hh or hh:mm or hh:mm:ss
  - Categorize all hidden columns with a submenu
  - Position the header context menu to the right border of the column (to see the content of the column)
  
  
* Tour Book View / Tour Editor

  - Set cadence format to rpm/spm

	 * 1.0 = Revolutions per minute (RPM)
	 * 2.0 = Steps per minute (SPM)

* FIT Import

  * Set cadence format to 2.0 when a stride sensor is detected
  

* Tour Editor - Info Tab

  - Show stride sensor state

	 
* Tour Chart
 
  - Show cadence with the correct format
  - Replaced chart options drop-down menu with a slideout


* Tour Book View

  - New context menu actions for the selected tours
   
    * Adjust tour start temperature values
    * Set cadence format for selected tours


* Tour Tooltip

  - Use value format from the preferences
